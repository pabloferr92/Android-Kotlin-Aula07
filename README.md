Android Kotlin - Aula 07

Este repositório contém o projeto desenvolvido durante a Aula 07 do curso de desenvolvimento Android com Kotlin. O objetivo da aula é praticar conceitos básicos de desenvolvimento de aplicações Android utilizando a linguagem Kotlin e o Android Studio.

📱 Sobre o Projeto

O projeto consiste em uma aplicação Android simples, utilizada para demonstrar a estrutura básica de um aplicativo Android e o funcionamento de uma Activity principal.

No Android, a MainActivity é responsável por controlar a lógica da tela principal do aplicativo e gerenciar interações com a interface do usuário.

Durante a aula, são explorados conceitos fundamentais como:

Estrutura de um projeto Android

Uso da classe AppCompatActivity

Método onCreate

Definição do layout com setContentView

Interação entre código Kotlin e layout XML

🛠️ Tecnologias Utilizadas

Kotlin

Android Studio

Android SDK

XML para layouts

📂 Estrutura do Projeto
app/
 ├── src/
 │   └── main/
 │       ├── java/com/example/projetoaula07/
 │       │   └── MainActivity.kt
 │       ├── res/
 │       │   └── layout/
 │       │       └── activity_main.xml
 │       └── AndroidManifest.xml
Arquivos principais

MainActivity.kt
Contém a lógica principal da aplicação e o ponto de entrada da interface.

activity_main.xml
Define a interface gráfica da tela principal do aplicativo.

AndroidManifest.xml
Arquivo responsável por declarar componentes da aplicação e configurações essenciais.

▶️ Como Executar o Projeto

Clone este repositório:

git clone https://github.com/pabloferr92/Android-Kotlin-Aula07.git

Abra o projeto no Android Studio

Aguarde o download das dependências do Gradle.

Execute o projeto em:

um emulador Android, ou

um dispositivo físico conectado
